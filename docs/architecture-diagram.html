🔄 Typical Data Flow

1
User enters transaction in React form (date, description, net amount, VAT rate, category)
2
Frontend calculates VAT and Gross automatically and displays preview
3
API POST request sent to backend with transaction data
4
Backend validates and inserts data into SQLite database
5
Database file updated and saved to Google Drive sync folder
6
Google Drive syncs the updated database to cloud automatically
7
Frontend refreshes transaction list to show the new entry
